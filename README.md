# Traffic Vision

Traffic camera object detection and tracking implementation using YOLO models.

## Project Status

Clean repository restart after 3-4 months of development work became fragmented across multiple environments.

## Environment Setup

```bash
conda env create -f environment.yml
conda activate traffic-vision
```

## Dependencies

**Conda packages:**
- jupyter, opencv, pytorch, torchvision, numpy, pandas, matplotlib

**Pip packages:**
- ultralytics (YOLOv8)
